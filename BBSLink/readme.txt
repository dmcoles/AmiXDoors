Setup for AmiExpress (Requires v5.4.0 as a minimum)

1. Sign up for an account at http://www.bbslink.net/
2. Update the bbslink.cfg with your syscode, authcode and schemecode.
3. Copy the contents of the doors folder to your bbs:doors folder.
4. Copy any or all of the files from the commands folder to bbs:commands

LINKMENU - brings up the menu of games
LINKWALL - view or write on the bbslink shared wall
LORD - legend of the red dragon
LRD2 - legend of the red dragon II: New World
MZKL - MZK LORD

When running the command from Ami-Express you can also pass in the 4 digit
code as a parameter to the command so for example LINKMENU LUNA will run
Lunatix.
